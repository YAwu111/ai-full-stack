# 《Web前端基础》课程设计报告（题目5：企业官网）

## 1. 项目概述
- 项目名称：智汇科技企业官网
- 技术栈：HTML5 + CSS3 + JavaScript（无后端、无数据库）
- 页面规模：11页（满足不少于10页要求）

## 2. 功能实现
- 页面导航与响应式布局（PC / 平板 / 手机）
- 首页轮播（自动切换 + 手动控制）
- 产品中心（分类筛选、关键词搜索、分页）
- 产品详情（URL参数读取、数量调整、加入购物车本地模拟）
- 新闻中心（分类筛选、新闻详情联动）
- 表单验证（联系我们、招聘投递）
- FAQ 折叠交互

## 3. 项目结构
- `index.html`：首页
- `about.html`：关于我们
- `services.html`：服务中心
- `products.html`：产品中心
- `product-detail.html`：产品详情
- `cases.html`：客户案例
- `news.html`：新闻中心
- `news-detail.html`：新闻详情
- `careers.html`：人才招聘
- `contact.html`：联系我们
- `faq.html`：常见问题
- `css/style.css`：统一样式
- `js/app.js`：统一交互逻辑

## 4. 关键技术说明
- `position: sticky` 实现吸顶导航
- `CSS Grid` 与媒体查询实现响应式布局
- `localStorage` 模拟购物车数量持久化
- `URLSearchParams` 读取详情页参数
- 原生 JS 完成筛选、分页、表单验证、手风琴

## 5. 测试与优化
- 页面链接完整性测试
- 手机端断点（760px、980px）布局测试
- 表单边界值测试（空值、邮箱格式、手机号格式）
- 交互性能优化（按需初始化模块、事件绑定复用）

## 6. 总结
本项目完整覆盖企业官网常见模块与核心交互，达到课程设计对页面数量、界面规范、交互逻辑与代码结构的要求。
