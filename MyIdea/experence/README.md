# 企业官网课程设计（题目5）

## 项目简介
本项目为《Web前端基础》课程设计题目5（企业官网），采用纯前端技术实现：
- HTML5 + CSS3 + JavaScript
- 无后端、无数据库
- 响应式布局（PC/平板/手机）

## 页面清单（11页）
1. `index.html` 首页
2. `about.html` 关于我们
3. `services.html` 服务中心
4. `products.html` 产品中心
5. `product-detail.html` 产品详情
6. `cases.html` 客户案例
7. `news.html` 新闻中心
8. `news-detail.html` 新闻详情
9. `careers.html` 人才招聘
10. `contact.html` 联系我们
11. `faq.html` 常见问题

## 已实现交互
- 轮播图自动播放与手动切换
- 产品分类筛选、搜索、分页
- 产品详情页数量加减与本地购物车计数
- 新闻分类筛选与详情联动
- 留言表单与招聘表单校验
- FAQ 折叠展开
- 移动端菜单切换

## 运行方式
直接双击 `index.html` 或使用任意静态服务器打开 `MyIdea/experence` 目录。
