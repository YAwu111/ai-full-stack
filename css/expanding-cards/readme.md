# 弹性布局

- 开关标签
- HTML文档
    一些html片段,html文档流
    默认从上到下(块级),从左到右(行内)，像水流一样流满整个页面.
- 行内元素/块级元素
    display:inline; **不可以设置宽高**
    display:block; 块级元素**可以设置宽高**,适合做容器(**但是容器之间默认从上到下排列**)
    display:inline-block; 适合做布局 但由于换行符的存在,导致行内块元素之间存在间隙
    有没有别的方案如何实现多列形式的布局？

- 布局 css提供
    两列式布局 切换display inline-block
